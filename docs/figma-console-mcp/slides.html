1 · Intro

How is this different from Figma’s own MCP?

  • Figma’s MCP — strong at design → code and Dev Mode workflows
  • Console MCP — built for design-system work in Figma: tokens, variants, FigJam, audits, creating on the canvas

You can use both. Today we’re here for getting more done in Figma.
