Why

Less archaeology, more answers

  • “What tokens does Button use?”
  • “Summarize variants for handoff”
  • “Anything off in this frame?”

What it is

Claude ↔ Figma, through a connector

MCP = a plug-in API. You chat; Claude calls Figma tools instead of guessing.

Today: read-only setup — no Node, no plugin.

Setup

Three steps

  1. Claude Desktop → Settings → Connectors
  2. Add connector — Name: Figma Console
  3. URL: https://figma-console-mcp.southleft.com/sse

First use: authorize Figma in the browser. Restart Claude if needed.

Prompt tips

Keep it concrete

Do

  • Paste the Figma URL
  • Name the component
  • Ask for a summary, not a rewrite

Don’t

  • Expect it to edit Figma (read-only)
  • Ask it to publish to the library
  • Skip the URL

Later

Need write access?

Creating frames, editing variables, or token sync needs Node + Figma Desktop + Bridge plugin. Pair with eng — see the optional section in the handout.
